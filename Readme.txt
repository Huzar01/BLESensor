BLESensor - Basic BLE sensor, send it any value to turn on motor/sensor
BLEPasswd - Send password to sensor to turn it on 
BLEWrite - Allows multiple write data via BLE. EX: - run sensor for 5 rotations...10...etc..(Work in progress)
